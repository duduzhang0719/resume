# 张雪娇的简历

## 文件说明
- `resume.html`: 主要简历文件，可直接用浏览器打开
- `profile.jpg`: 头像图片（可选）

## 使用说明
1. 直接用浏览器打开 `resume.html` 文件即可查看简历
2. 如需打印，请使用 Chrome 浏览器，并在打印设置中：
   - 勾选"背景图形"选项
   - 选择 A4 纸张大小
   - 边距选择"无"或"最小"

## 浏览器兼容性
推荐使用以下浏览器查看：
- Chrome (推荐)
- Firefox
- Safari
- Edge

## 注意事项
- 如果看不到头像图片，这是正常的，不影响简历的整体展示
- 简历采用响应式设计，可以适应不同尺寸的屏幕
- 打印时会自动移除背景图案，呈现更适合打印的版本
